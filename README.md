AI Schema Selector is a web application that converts natural language queries into accurate SQL statements for SQLite databases. It uses Llama 3 to analyze database schemas, correct spelling errors, and generate optimized queries. Users can upload a .db file, input a query in plain English, and get a properly formatted SQL statement. The app also allows users to execute queries and view results directly.

Tech Stack: Flask, SQLite, Ollama (Llama 3), HTML, CSS, JavaScript.

🚀 Features:
✔ Upload and analyze SQLite databases
✔ AI-powered NL-to-SQL conversion
✔ Spelling correction & schema understanding
✔ Execute queries within the app
